1. Why is textContent safer than innerHTML?

innerHTML inserts the text as HTML, with the tags, and textContent inserts it as text, with all symbols treated literally.

Additionally, if you were to use innerHTML to get input from somebody else, they could add in malicious code to the site.


2. How are HTML elements connected to JavaScript variables using DOM selection?
